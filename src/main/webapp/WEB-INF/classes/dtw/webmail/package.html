Provides a kernel, the controllers and related classes for a Java webmail solution based on the model-view-controller (MVC) pattern. To deploy this package you will need to have a servlet container and a Java Server Pages (JSP) solution.
See related documentation for more information.

Package Specification

  • Based on Java Servlet Specification 2.0
  • Based on JavaMail 1.1 and the Java Activation Framework(JAF) 1.0
    (See related documentation for more information and legal notices.)
  • Uses Castor XML and Apache Xerces-J for persistency.
    (See related documentation for more information and legal notices.)
  • Uses Jakarta-ORO for text processing (regular expressions package).
    (See related documentation for more information and legal notices.)
